Setup & Configuration

Follow the steps below to set up and configure the website:

  • Ensure you have a database server (MS SQL Server Express) set up and running.
  • Create new database named FoodOrderingDB (You can user SSMS 20 for best compatibility).
  • Import the provided database schema file (FastOrder.sql) into the database server.
  • Edit the connection string in the configuration file (Web.config) to match the database credentials. (If you name the database correctly, then there should no need to do this step).
  • Open the project on Visual Studio 2022 and open Solution Explorer.
  • In the User folder, right click the Default.aspx file, and set it to Set As Start Page.
  • Access the website by running the project.

For further assistance, contact the developer or refer to the documentation provided in the project files.
